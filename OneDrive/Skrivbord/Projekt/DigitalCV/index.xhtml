DigatalCV - ett digitaltCV karriärinformation där
studenter kan skicka digital-CV till arbetsgivare

I DigitalCV kan du som student komma att skapa, hantering och delning CV-pdf online med fokus säkerhet och skicka till arbetsgivare med engångskoder till arbetsgivare.



För dig som är student

DigitalCV erbjuder dig att skapa CV med karriärinformationer.

På DigitalCV kan du på ett enkelt sätt skapa CV:

  • Ladda upp bild
  • Kontakt uppgifter
  • Link GitHub
  • profil
  • Personbrev
  • Utbildning och betyg
  • Arbetslivserfaranhet
  • Programfardighet

Med skapa CV, du kommer att hämta pdf-fil länk med engångskod och du kan skicka länken till arbtesgivare. Arbetsgivare kan åtkomma till delade pdf-länkar med engångskod. Logga in med ditt konto och skriva din uppgifter
